3. For Linear Regression, please upload only files that are of ".csv" format. Do not upload files of formats other than the mentioned. The model uses the last column of the uploaded data as target class and the remaining columns as dependent variables or features. So make sure your target class is at the last column. Please clean the data and handle missing or null values before uploading. This model is suitable for supervised data only.
